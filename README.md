# 🤖 My AI Agent Project

基于 **OpenClaw** + **Xiaomi MiMo API** 搭建的个人 AI Agent 工作流。

## 项目目标

构建一个能自动化处理日常任务的智能助手系统，探索 AI Agent 在个人效率领域的应用。

## 技术栈

- **Agent 框架：** [OpenClaw](https://github.com/openclaw/openclaw)
- **核心模型：** Xiaomi MiMo-V2.5-Pro（Agent & Coding 优化）
- **多模态：** MiMo-V2-Omni（图像、音频理解）
- **推理框架：** SGLang / vLLM

## 当前进度

- [x] OpenClaw 环境部署
- [x] 基础 Agent 配置
- [ ] MiMo API 接入调试
- [ ] 多工具调用链路（文件管理、Web 搜索、日程）
- [ ] 个人知识管理 Agent

## 后续规划

1. **知识管理 Agent** — 自动整理和检索笔记，基于 RAG 实现个人知识库
2. **多模态 Agent** — 接入 MiMo-V2-Omni，实现图像理解 + 语音交互
3. **开源分享** — 将 Agent 工作流配置和开发经验开源
